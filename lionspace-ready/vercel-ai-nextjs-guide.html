🌟 סקירה כללית

Vercel AI SDK הוא טולקיט TypeScript מתקדם המיועד לבניית אפליקציות מופעלות בינה מלאכותית. הפרויקט מתמחה בשילוב חלק עם Next.js ומספק כלים מקצועיים לפיתוח אפליקציות AI ברמת production.

💡 למה לבחור ב-Vercel AI SDK?
הפרויקט מציע שילוב מושלם בין קלות שימוש לבין יכולות מתקדמות, עם תמיכה נרחבת ב-Next.js ובמגוון רחב של מודלים וספקי AI.

📦 חבילות ייעודיות ל-Next.js

החבילות הראשיות:

  • @ai-sdk/react - Hooks מתקדמים המיועדים במיוחד ל-React ו-Next.js
  • @ai-sdk/rsc - תמיכה מלאה ב-React Server Components
  • ai - החבילה הראשית עם תמיכה נרחבת ל-Next.js
⚠️ הערה חשובה:
כדי לנצל את מלא היכולות, מומלץ להשתמש ב-Next.js 13+ עם App Router לקבלת התמיכה הטובה ביותר ב-Server Components.

🚀 הקמת פרויקט Next.js חדש

יצירת פרויקט:

# יצירת פרויקט Next.js חדש pnpm create next-app@latest my-ai-app # התקנת החבילות הנדרשות pnpm add ai @ai-sdk/react @ai-sdk/openai zod # חבילות נוספות לפיצרים מתקדמים pnpm add @ai-sdk/anthropic @ai-sdk/google @ai-sdk/mistral
💡 טיפ:
השתמש ב-TypeScript לחוויה טובה יותר. ה-SDK מגיע עם type definitions מלאים.

💻 יישום בסיסי - מערכת צ'אט

1. יצירת API Route (App Router):

// app/api/chat/route.ts import { openai } from '@ai-sdk/openai'; import { streamText, UIMessage, convertToModelMessages } from 'ai'; export async function POST(req: Request) { const { messages }: { messages: UIMessage[] } = await req.json(); const result = streamText({ model: openai('gpt-4o'), messages: convertToModelMessages(messages), temperature: 0.7, maxTokens: 1000, }); return result.toUIMessageStreamResponse(); }

2. קומפוננטת Client:

🔥 פיצרים מתקדמים

1. Tools ו-Tool Calling:

// tools/weather.ts import { tool } from 'ai'; import { z } from 'zod'; export const weatherTool = tool({ description: 'Get weather information for a specific location', inputSchema: z.object({ location: z.string().describe('The city or location'), unit: z.enum(['celsius', 'fahrenheit']).default('celsius'), }), execute: async ({ location, unit }) => { // קריאה לAPI מזג אוויר אמיתי const temperature = 22 + Math.floor(Math.random() * 21) - 10; return { location, temperature: unit === 'fahrenheit' ? (temperature * 9/5) + 32 : temperature, unit, description: 'בהיר עם עננים קלים' }; } }); // שימוש ב-Route: import { generateText } from 'ai'; const result = await generateText({ model: openai('gpt-4'), tools: { weather: weatherTool }, prompt: 'מה מזג האוויר בתל אביב?' });

2. Structured Data Generation:

// utils/structured-generation.ts import { generateObject } from 'ai'; import { z } from 'zod'; const recipeSchema = z.object({ recipe: z.object({ name: z.string(), ingredients: z.array(z.string()), instructions: z.array(z.string()), prepTime: z.number(), servings: z.number() }) }); export async function generateRecipe(dishType: string) { const { object } = await generateObject({ model: openai('gpt-4'), schema: recipeSchema, prompt: `צור מתכון מפורט ל${dishType}` }); return object.recipe; }

3. Streaming Capabilities:

// utils/streaming.ts import { streamText } from 'ai'; export async function* streamStory(prompt: string) { const { textStream } = streamText({ model: openai('gpt-4'), prompt, temperature: 0.8, }); for await (const textPart of textStream) { yield textPart; } } // שימוש ברכיב: 'use client'; import { useState } from 'react'; export function StoryGenerator() { const [story, setStory] = useState(''); const generateStory = async () => { setStory(''); for await (const part of streamStory('כתוב סיפור על אבירים')) { setStory(prev => prev + part); } }; return ( <div> <button onClick={generateStory}>צור סיפור</button> <div style={{whiteSpace: 'pre-wrap'}}>{story}</div> </div> ); }

4. Multi-modal (טקסט + תמונות):

// utils/vision.ts import { generateText } from 'ai'; export async function analyzeImage(imageUrl: string, question: string) { const result = await generateText({ model: openai('gpt-4-vision-preview'), messages: [ { role: 'user', content: [ { type: 'text', text: question }, { type: 'image', image: imageUrl } ] } ] }); return result.text; }

🏭 אדריכלות Production

Rate Limiting עם Upstash:

// lib/rate-limit.ts import { Ratelimit } from '@upstash/ratelimit'; import { Redis } from '@upstash/redis'; const ratelimit = new Ratelimit({ redis: Redis.fromEnv(), limiter: Ratelimit.slidingWindow(10, "10 s"), }); // שימוש ב-API Route: export async function POST(request: Request) { const ip = request.headers.get('x-forwarded-for') || 'anonymous'; const { success } = await ratelimit.limit(ip); if (!success) { return new Response('Rate limit exceeded', { status: 429 }); } // המשך הלוגיקה... }

Telemetry ומעקב:

// lib/telemetry.ts import { telemetry } from 'ai'; telemetry.setProviders({ 'openai': { apiKey: process.env.OPENAI_API_KEY!, onStart: (params) => { console.log('Starting AI request:', params); }, onFinish: (result) => { console.log('AI request finished:', result); }, onError: (error) => { console.error('AI request failed:', error); } } });

Error Handling מתקדם:

// hooks/useErrorHandling.ts import { useState } from 'react'; export function useErrorHandling() { const [error, setError] = useState<string | null>(null); const handleError = (error: any) => { if (error.code === 'RATE_LIMIT_EXCEEDED') { setError('יותר מדי בקשות. נסה שוב בעוד מספר דקות.'); } else if (error.code === 'INSUFFICIENT_CREDITS') { setError('אזל הקרדיט. אנא עדכן את התוכנית שלך.'); } else { setError('שגיאה לא צפויה. נסה שוב מאוחר יותר.'); } }; return { error, handleError, clearError: () => setError(null) }; }
